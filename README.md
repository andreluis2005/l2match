
# Which L2 Are You?

**Link para o App:** [https://l2match.vercel.app](https://l2match.vercel.app)

**Built with 🛠️ MiniKit + Base**

Many users struggle to understand and connect with Layer 2 blockchains due to a lack of engaging educational tools.

**“Which L2 Are You?”** is a fun and interactive quiz that matches users to an Ethereum Layer 2 (L2) network — Base, Arbitrum, Optimism, or zkSync — based on their preferences and personality traits.

## 🌟 Features

- **Personality-based L2 matching**
- **Built with Base + MiniKit**
- **Save results onchain (Base)**
- **Share results on Warpcast (Farcaster)**
- **Beautiful UI/UX, responsive and user-friendly**

## 🧠 Why it matters

L2s can be complex. This app helps break down barriers to adoption by making learning about L2s engaging and personal. Users discover which L2 matches their values — whether it’s tech innovation, low fees, decentralization, or community.
